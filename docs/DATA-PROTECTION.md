# Data Protection Notes

This system is designed to help a club manage member information responsibly.

## Built-in controls

- role-based access
- member self-service profile updates
- consent fields
- internal directory opt-in
- audit logging
- private upload storage outside web root
- member export
- GDPR/data retention policy pages

## Recommended policy areas

The club should maintain written policies covering:

- what personal data is collected
- why it is collected
- who can access it
- how long it is retained
- how members can request correction/deletion/export
- how backups are protected
- how incidents are handled

## Directory

The internal directory only displays members who opt in.

It should show:

- name
- callsign

No address, phone number, payment information, emergency contact, notes or private data should be shown.
