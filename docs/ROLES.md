# Roles and Permissions

The system supports multiple roles per user.

Common roles:

- Member
- Committee Member
- Member DB User
- Brickworks Reviewer / Manager
- Admin

## Rules

- Members can view/edit their own profile.
- Committee members can manage club operations such as events, equipment, attendance and actions.
- Member DB users can access the membership database.
- Admins can manage users, roles, system settings and protected admin functions.
- Admin accounts cannot be deleted by non-admins.
- The system prevents there being fewer than one active admin.

## Role changes

Admin users can assign/change user roles from the Users page.

All role changes are written to:

- role history
- audit log
